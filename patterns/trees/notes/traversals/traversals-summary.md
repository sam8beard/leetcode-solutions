# Summary of Different Traversals

## Import Note
> == If you can visualize *where the root appears*, you already understand the traversal. ==

## Comparison

| Traversal     | Order Rule               | Root Visited |
|---------------|--------------------------|--------------|
| Inorder       | Left → Root → Right      | Middle       |
| Preorder      | Root → Left → Right      | First        |
| Postorder     | Left → Right → Root      | Last         |
| Level Order   | Level by level           | First        |


## Mental Model Summary

- **Inorder**: “Left first, then me”
- **Preorder**: “Me first”
- **Postorder**: “Me last”
- **Level Order**: “Row by row”
